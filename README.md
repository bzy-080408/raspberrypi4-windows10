# 树莓派WOR安装之后处理进程
# 使用方法（Chinese）：
 step0
 下载，，右键以管理员身份运行
 
 step1
 选择新建的NTFS分区盘符
 
 step2
 选择新建的FAT32分区盘符
 
 step3
 输入S经行部署镜像
 
 step4
 待应用更改完成之后输入E退出
